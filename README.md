### Run Locally

To run **Futras** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/KhushbuChauhan-06/Fitness.git
```

Windows:

```bash
git clone https://github.com/KhushbuChauhan-06/Fitness.git
```
